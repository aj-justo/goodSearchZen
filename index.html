goodSearchZen by ajweb

Zen Cart search results ordered by relevance.

On a default Zen Cart installation, if you search for any keyword, say 'book', you get the results ordered alphabetically, by price, or other fields, but not by relevance, which of course, it is the most important order when you are searching. It is very usual to get no relevant results in the first page due to this lack of relevancy search.

Dependencies

Zen Cart 1.3.8 or higher. MySQL 4.3 or higher. PHP 4 or higher (though only tested so far with PHP 5.x versions)

Install

It only uses a couple of admin side files for configuration and an observer on the catalog side, no need to edit or override any file. Just upload to the proper directories and activate on admin page.
